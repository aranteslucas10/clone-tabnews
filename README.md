# clone-tabnews
Aprimorando meus conhecimentos com a implementação do https://www.tabnews.com.br feita no https://curso.dev

## Dependências
* nodejs lts/hydrogen
* next@13.1.6
* react@18.2.0
* react-dom@18.2.0

## Deploy Site Estático com o GitHub Pages
### Gerar Site Estático
```bash
npm run build
npm run export
```
